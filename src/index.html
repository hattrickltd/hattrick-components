Hattrick Components

This page shows of the stencil components and various ways to use them.

Bars and Avatars are lazy loaded. If you look in the networks tab and then click on "load bars" or "load avatars" you'll see that new files are loaded automatically. There's no manual work for this, it happens as soon as a <hattrick-bar> or <hattrick-avatar> is added to the DOM.

Match clock

Regular match clocks. They start counting backwards during halftime and overtime breaks.

Attributes

matchtime: Date | string | number

Examples

Timers

These are basic timers. They count down to 00:00:00, and optionally start counting upwards after reaching 0. If time is too far from 0, it can optionally just count days instead of hours:minutes:seconds.

Attributes

daysText: string = "days" deadline: Date | string | number keepCounting: boolean = false maxHours: number = 72

Examples

Progress arcs

These are progress arcs. They're useful to show anything between 0% and 100%.

Attributes

size: number complete: number [0-1]

Css variables

--progress-arc-stroke-width: 8px --progress-arc-color: #00aa00 --progress-arc-rest-color: transparent | #000000

Examples

You can use predefined size: size="large" (same as size="44")*


Or: size="small" (same as size="29")*


Or you can define yourself size="29". Note however how the second stamina bar and font doesn't adjust with manual sizes. You have to manually set font-size and --progress-arc-stroke-width.

Flip

The flip component allows us to have a component with both a front and a back which we can flip.

Attributes

flipped: boolean direction: "x" | "y"

Css variables

--flip-height: 100%; --flip-width: 100%; --flip-duration: 1s; Note: At the very minimum you will want to set height for direction="x" or width for direction="y"

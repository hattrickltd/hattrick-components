Hattrick Components

This page shows of the stencil components and various ways to use them.

Bars and Avatars are lazy loaded. If you look in the networks tab and then click on "load bars" or "load avatars" you'll see that new files are loaded automatically. There's no manual work for this, it happens as soon as a <ht-bar> or <ht-avatar> is added to the DOM.

Timers

These are basic timers. They count down to 00:00:00, and optionally start counting upwards after reaching 0. If time is too far from 0, it can optionally just count days instead of hours:minutes:seconds.

Attributes

daysText: string = "days"
deadline: Date | string | number
keepCounting: boolean = false
maxHours: number = 72

Progress arcs

These are progress arcs. They're useful to show anything between 0% and 100%.

Attributes

size: number
complete: number [0-1]

Css variables

--progress-arc-stroke-width: 8px
--progress-arc-color: #00aa00
--progress-arc-rest-color: transparent | #000000
